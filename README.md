# 🛠️ DigiTools Platform

> A modern digital tools marketplace where professionals discover, compare, and purchase the best software to supercharge their workflow.

---

## 🚀 Live Site

[🔗 Live Demo](https://safin313-stack.github.io/B13-A6-DigiTools-Platform/)

---

## 🧰 Technologies Used

- **React.js** — Component-based UI
- **Vite** — Lightning-fast build tool
- **Tailwind CSS** — Utility-first styling
- **DaisyUI** — Tailwind component library
- **React-Toastify** — Toast notifications
- **JSON** — Product data management

---

## ✨ Key Features

1. **Smart Cart System** — Add products to cart, remove items individually, view running total, and clear everything on checkout — all with toast notifications.

2. **Product/Cart Toggle View** — A clean tab system lets users switch between browsing all products (3-column grid) and managing their cart (1-column list) with live cart count badge.

3. **Responsive & Polished UI** — Fully responsive dark-themed UI with animated hero, floating cards, gradient accents, and smooth hover interactions.

## 📦 Getting Started

```bash
git clone https://github.com/Safin313-stack/B13-A6-DigiTools-Platform.git
cd digitools
npm install

```

## 📁 Project Structure

```
src/
├── components/
│   ├── Navbar.jsx        # Sticky navbar with cart count badge
│   ├── Banner.jsx        # Hero section with floating card
│   ├── Stats.jsx         # Platform statistics
│   ├── MainSection.jsx   # Products grid + cart toggle
│   ├── ProductCard.jsx   # Individual product card
│   ├── CartItem.jsx      # Cart item with remove button
│   ├── Steps.jsx         # How it works section
│   ├── Pricing.jsx       # Pricing plans
│   └── Footer.jsx        # Site footer
├── products.json         # 8 product entries
├── App.jsx               # Root with cart state
└── index.css             # Global styles + animations
```
